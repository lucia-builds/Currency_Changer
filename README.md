React Currency Converter
A high-performance, real-time currency conversion application built with React, Tailwind CSS, and Vite. This project demonstrates advanced state management, custom hook architecture, and seamless API integration.
------------------------------
🚀 Key Features

* Live Exchange Rates: Integration with the @fawazahmed0/currency-api for 150+ global currencies.
* Custom Hook Architecture: Encapsulated data-fetching logic within a reusable useCurrencyInfo hook.
* Instant Swap Logic: Synchronized state management to flip "From" and "To" currencies and amounts instantly.
* Dynamic UI: Responsive, accessible components styled with Tailwind CSS for a modern user experience.

🛠️ Technical Stack

* Library: React.js (Functional Components & Hooks)
* Styling: Tailwind CSS (Utility-first CSS)
* API Handling: Fetch API with optimized useEffect dependency tracking.
* Build Tool: Vite (For lightning-fast HMR and optimized builds)

⚙️ Installation & Setup

   1. Clone the repository:
   
   git clone "repository_github_Link"
   
   2. Install dependencies:
   
   npm install
   
   3. Run the development server:
   
   npm run dev
